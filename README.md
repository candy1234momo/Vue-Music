> vue-music

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit vue-music
npm run unit

# run e2e vue-music
npm run e2e

# run all vue-music
npm test
```
